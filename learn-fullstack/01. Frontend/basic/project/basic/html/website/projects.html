Fase 1 — HTML

Toko Sembako Maju Jaya

Januari 2026 — sekarang

Sistem informasi toko sembako lengkap: katalog produk dengan kartu artikel, form kasir dengan validasi aksesibel, tabel laporan stok, dan daftar supplier. Dibangun dengan struktur file terpisah (HTML, CSS eksternal, JS eksternal) mengikuti standar industri.

  • Semantic HTML5 + layout semantik penuh
  • Accessibility: skip link, ARIA, focus management
  • SEO: meta tags, Open Graph, JSON-LD structured data
  • External CSS dengan CSS variables
  • JavaScript dengan defer dan role="alert"
