13 · Pseudo Elements

Pseudo Element vs Pseudo Class

Pseudo class (:) menarget elemen berdasarkan state. Pseudo element (::) membuat bagian virtual dari elemen yang tidak ada di HTML tapi bisa di-style atau ditambahkan konten. Ditulis dengan dua titik dua untuk membedakan dari pseudo class.

⚠️ Wajib ada properti content (bisa kosong "") pada ::before dan ::after agar muncul di halaman.

::before dan ::after — Konten Dekoratif

Dua pseudo element paling sering dipakai. ::before memasukkan konten virtual sebelum konten asli elemen. ::after memasukkan sesudah. Keduanya berperilaku seperti inline element secara default tapi bisa diubah dengan display.

Produk kami sudah diuji kualitasnya secara ketat.
Beras Premium 5kg — ::after untuk badge DISKON 
.kartu-produk {
  position: relative; /* diperlukan agar ::after bisa absolute */
}

/* Badge "Diskon" di sudut kanan atas — tidak perlu HTML tambahan */
.kartu-produk.diskon::after {
  content: "DISKON";    /* WAJIB ADA — tanpa ini elemen tidak muncul */
  position: absolute;
  top: 8px;
  right: 8px;
  background: #c62828;
  color: white;
  font-size: .7rem;
  padding: 2px 8px;
  border-radius: 4px;
}

/* Garis dekoratif sebelum heading */
h2::before {
  content: "";         /* kosong tapi tetap WAJIB ada */
  display: block;
  width: 40px;
  height: 3px;
  background: #2e7d32;
  margin-bottom: 8px;
}

/* Icon setelah link eksternal */
a[target="_blank"]::after {
  content: " ↗";
  font-size: .8em;
  opacity: .6;
}

::first-line dan ::first-letter

/* Baris pertama paragraf */
p::first-line {
  font-weight: bold;
  color: #2e7d32;
}

/* Huruf pertama — efek drop cap seperti majalah */
article > p:first-child::first-letter {
  font-size: 3rem;
  font-weight: bold;
  float: left;
  line-height: 0.8;
  margin-right: 8px;
  color: #2e7d32;
}

::placeholder dan ::selection

/* Teks placeholder di dalam input */
input::placeholder {
  color: #9e9e9e;
  font-style: italic;
}

/* Teks yang diseleksi (di-highlight) pengguna */
::selection {
  background: #a5d6a7;
  color: #1b5e20;
}
💡 Prinsip untuk ::before dan ::after: gunakan hanya untuk konten dekoratif yang tidak punya makna semantik. Kalau konten penting untuk dipahami (label, informasi), taruh di HTML. Screen reader tidak membaca pseudo elements secara konsisten.
