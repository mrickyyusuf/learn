Flexbox vs Grid: Kapan Pakai Mana?

Flexbox dirancang untuk layout satu dimensi — baris ATAU kolom. Grid dirancang untuk layout dua dimensi — baris DAN kolom sekaligus. Aturan praktis: gunakan Flexbox untuk komponen kecil (navigasi, tombol group, satu baris kartu), gunakan Grid untuk layout halaman atau area yang butuh kontrol atas baris DAN kolom.

Kenyataannya keduanya sering dipakai bersama: Grid untuk struktur besar halaman, Flexbox untuk susunan elemen di dalam setiap cell Grid.

Flexbox — 1D Layout

Properti Container (Parent)

/* Aktifkan flex container */
.container {
  display: flex;

  /* Arah — default: row (kiri ke kanan) */
  flex-direction: row;         /* → → → */
  flex-direction: column;      /* ↓ ↓ ↓ */
  flex-direction: row-reverse; /* ← ← ← */

  /* Sumbu utama (main axis) — justify-content */
  justify-content: flex-start;    /* kiri (default) */
  justify-content: flex-end;      /* kanan */
  justify-content: center;        /* tengah */
  justify-content: space-between; /* tersebar, ujung-ujung menempel */
  justify-content: space-around;  /* tersebar, ada ruang di ujung */
  justify-content: space-evenly;  /* semua jarak sama */

  /* Sumbu silang (cross axis) — align-items */
  align-items: stretch;    /* memenuhi tinggi container (default) */
  align-items: flex-start; /* atas */
  align-items: center;     /* tengah secara vertikal */
  align-items: flex-end;   /* bawah */

  /* Wrap: izinkan item pindah ke baris baru jika tidak cukup */
  flex-wrap: nowrap;  /* default: semua satu baris */
  flex-wrap: wrap;    /* item bisa pindah ke baris berikutnya */

  gap: 16px;          /* jarak antar item */
  gap: 16px 8px;      /* row-gap column-gap */
}

Properti Item (Child)

.item {
  /* flex: grow shrink basis — shorthand */
  flex: 1;         /* bisa mengembang dan menyusut secara merata */
  flex: 0 0 200px; /* lebar tetap 200px, tidak tumbuh, tidak menyusut */
  flex: 1 0 auto;  /* tumbuh sebanyak perlu, tidak menyusut, basis auto */

  /* Kontrol individual */
  flex-grow: 1;    /* seberapa banyak ia tumbuh relatif terhadap sibling */
  flex-shrink: 0;  /* 0 = tidak menyusut saat container mengecil */
  flex-basis: 200px; /* ukuran awal sebelum grow/shrink diterapkan */

  align-self: center; /* override align-items hanya untuk item ini */
  order: -1;          /* ubah urutan visual tanpa mengubah HTML */
}

CSS Grid — 2D Layout

Properti Container

.container {
  display: grid;

  /* Definisi kolom */
  grid-template-columns: 200px 1fr 1fr;     /* kolom 1: 200px, kolom 2-3: sama */
  grid-template-columns: repeat(3, 1fr);    /* 3 kolom sama lebar */
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); /* responsif otomatis */

  /* Definisi baris */
  grid-template-rows: auto 1fr auto;        /* header, konten fleksibel, footer */

  /* Gap */
  gap: 16px;
  column-gap: 24px;
  row-gap: 16px;

  /* Named areas — layout halaman yang sangat readable */
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

Properti Item

.item {
  /* Span — item mencakup beberapa kolom atau baris */
  grid-column: span 2;           /* mencakup 2 kolom */
  grid-column: 1 / 3;            /* dari kolom 1 ke kolom 3 (exclusive) */
  grid-row: span 2;              /* mencakup 2 baris */

  /* Ditempatkan di named area */
  grid-area: header;
  grid-area: sidebar;
}

Studi Kasus: Grid Produk Toko Sembako

Ini adalah pola yang akan kamu gunakan ribuan kali untuk aplikasi retail — grid kartu produk yang responsif tanpa media query:

Beras Premium 5kg

Rp 78.000

Gula Pasir 1kg

Rp 14.000

Minyak Goreng 2L

Rp 32.500

Tepung Terigu 1kg

Rp 12.000

/* Grid produk yang responsif tanpa satu media query pun */
.product-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: 1rem;
}
/* auto-fill: buat sebanyak mungkin kolom yang muat
   minmax(200px, 1fr): setiap kolom minimal 200px, maksimal 1fr
   Hasilnya: di desktop muncul 4 kolom, di tablet 2, di mobile 1 */
💡 Pola repeat(auto-fill, minmax(min, 1fr)) adalah salah satu trik CSS paling powerful — grid responsif sempurna tanpa media query. Shopee, Tokopedia, dan hampir semua e-commerce menggunakan variasi pola ini untuk grid produk mereka.
