18 · CSS Variables (Custom Properties)

Konsep: Design Tokens

CSS variables (secara resmi disebut Custom Properties) adalah cara mendefinisikan nilai yang bisa digunakan berulang kali di seluruh stylesheet. Bayangkan mereka seperti konstanta dalam pemrograman. Di aplikasi retail berskala besar, variabel CSS adalah fondasi dari design system — sistem yang memastikan konsistensi visual di seluruh platform. Saat warna brand berubah, kamu hanya mengubah satu variabel dan seluruh aplikasi ikut berubah.

Sintaks Dasar

/* Definisi: selalu dimulai dengan -- (dua tanda hubung) */
/* :root adalah selector untuk elemen html, scope global */
:root {
  /* Design tokens — nama yang deskriptif, bukan nama warna */
  --warna-brand: #2e7d32;
  --warna-brand-gelap: #1b5e20;
  --warna-brand-muda: #e8f5e9;
  --warna-bahaya: #c62828;
  --warna-teks: #1a2c3e;
  --warna-teks-halus: #757575;
  --warna-bg: #f4f7fc;
  --warna-bg-kartu: #ffffff;
  --warna-border: #e0e0e0;

  /* Spacing scale */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  --spacing-xl: 40px;

  /* Typography */
  --font-utama: Georgia, serif;
  --font-mono: 'Courier New', monospace;
  --ukuran-sm: 0.875rem;
  --ukuran-md: 1rem;
  --ukuran-lg: 1.25rem;
  --ukuran-xl: 1.5rem;

  /* Shadows */
  --shadow-sm: 0 2px 8px rgba(0,0,0,0.05);
  --shadow-md: 0 4px 16px rgba(0,0,0,0.08);
  --shadow-lg: 0 8px 32px rgba(0,0,0,0.12);

  /* Border radius */
  --radius-sm: 4px;
  --radius-md: 8px;
  --radius-lg: 16px;
  --radius-full: 9999px;

  /* Transitions */
  --transisi: 0.2s ease;
}

/* Penggunaan dengan var() */
.kartu-produk {
  background: var(--warna-bg-kartu);
  border: 1px solid var(--warna-border);
  border-radius: var(--radius-md);
  padding: var(--spacing-md);
  box-shadow: var(--shadow-sm);
  transition: box-shadow var(--transisi), transform var(--transisi);
}

.kartu-produk:hover {
  box-shadow: var(--shadow-md);
  transform: translateY(-2px);
}

/* Fallback: nilai default jika variabel tidak terdefinisi */
color: var(--warna-brand, #2e7d32);    /* pakai #2e7d32 jika --warna-brand tidak ada */

Scope Lokal dan Override

/* Variabel bisa di-scope ke komponen tertentu */
.btn {
  --btn-bg: var(--warna-brand);
  --btn-color: white;
  background: var(--btn-bg);
  color: var(--btn-color);
  padding: var(--spacing-sm) var(--spacing-md);
  border-radius: var(--radius-full);
}

/* Variasi tombol: override variabel lokal saja */
.btn-bahaya {
  --btn-bg: var(--warna-bahaya);  /* hanya override ini */
}

.btn-besar {
  --spacing-sm: 12px;              /* hanya context ini yang terpengaruh */
  --spacing-md: 24px;
}

Dark Mode dengan CSS Variables

/* Light mode (default) */
:root {
  --warna-bg: #f4f7fc;
  --warna-bg-kartu: #ffffff;
  --warna-teks: #1a2c3e;
  --warna-border: #e0e0e0;
}

/* Dark mode: hanya override variabel, semua komponen ikut otomatis */
@media (prefers-color-scheme: dark) {
  :root {
    --warna-bg: #121212;
    --warna-bg-kartu: #1e1e1e;
    --warna-teks: #e0e0e0;
    --warna-border: #333333;
  }
}

/* Atau via class untuk toggle manual */
[data-theme="dark"] {
  --warna-bg: #121212;
  --warna-bg-kartu: #1e1e1e;
}

Manipulasi via JavaScript

/* Baca nilai variabel CSS */
const style = getComputedStyle(document.documentElement);
const warnaUtama = style.getPropertyValue('--warna-brand').trim();

/* Ubah nilai variabel CSS dari JavaScript */
/* Berguna untuk theming dinamis atau animasi berbasis data */
document.documentElement.style.setProperty('--warna-brand', '#1565c0');

/* Contoh nyata: progress bar stok */
function updateStokBar(persentase) {
  document.querySelector('.stok-bar')
    .style.setProperty('--stok-persen', persentase + '%');
}
/* CSS: .stok-bar::after { width: var(--stok-persen, 100%); } */
💡 CSS variables adalah bagian inti dari design system. Shopee, Tokopedia, Gojek — semuanya menggunakan sistem variabel seperti ini (dengan berbagai nama: design tokens, theme variables, CSS custom properties). Pelajari ini dengan serius karena ini akan menjadi fondasi sistem desain yang kamu bangun.
