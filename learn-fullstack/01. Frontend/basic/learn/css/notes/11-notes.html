Z-Index & Stacking Context

Z-index mengontrol urutan tumpukan elemen di sumbu Z (kedalaman). Elemen dengan z-index lebih tinggi tampil di depan. Tapi ada syarat: z-index hanya bekerja pada positioned elements (position selain static) dan pada elemen flex/grid.

z-index: 1
z-index: 2
z-index: 3
/* Z-index hanya bekerja pada positioned elements */
.modal    { position: fixed; z-index: 1000; }
.overlay  { position: fixed; z-index: 999; }
.dropdown { position: absolute; z-index: 100; }
.navbar   { position: sticky; z-index: 50; }

/* Stacking context baru dibuat oleh:
   - position: relative/absolute/fixed/sticky + z-index selain auto
   - opacity < 1
   - transform, filter, will-change
   Elemen dalam stacking context yang sama dibandingkan bersama. */
⚠️ Jika z-index tidak bekerja seperti yang diharapkan, kemungkinan ada stacking context yang membatasi. Periksa apakah ancestor punya opacity < 1 atau transform yang membuat stacking context terpisah.

CSS Specificity — Ringkasan

/* Skor: (ID, Class/Attr/PseudoClass, Tag) */
p                   /* (0,0,1) */
.harga              /* (0,1,0) */  
#kasir              /* (1,0,0) */  ← satu ID mengalahkan 1000 class
.kartu .harga       /* (0,2,0) */
#kasir .kartu p     /* (1,1,1) */

/* Aturan: kolom kiri SELALU menang atas kolom kanan, apapun jumlahnya.
   Tidak ada carry-over. Ini bukan desimal biasa. */

/* Praktik terbaik: hindari ID selector dan !important di CSS produksi.
   Pertahankan specificity serendah mungkin agar mudah di-override. */

CSS Tables

table {
  width: 100%;
  border-collapse: collapse;  /* gabungkan border antar sel */
  /* border-separate (default): tiap sel punya border sendiri */
}

th, td {
  padding: 10px 12px;
  border: 1px solid #ddd;
  text-align: left;
  vertical-align: top; /* konten mulai dari atas sel */
}

thead th {
  background: #2e7d32;
  color: white;
  position: sticky; /* sticky header untuk tabel panjang */
  top: 0;
}

tbody tr:nth-child(even) td { background: #f5f5f5; } /* zebra striping */
tbody tr:hover td { background: #e8f5e9; }           /* row highlight */

/* Responsive table: bungkus dalam overflow-x: auto */
.table-wrapper { overflow-x: auto; }

CSS Lists

/* Hapus style bawaan (sering dilakukan di reset) */
ul, ol { list-style: none; padding: 0; }

/* Custom bullet dengan list-style */
ul { list-style-type: disc; }    /* • */
ul { list-style-type: square; }  /* ■ */
ol { list-style-type: decimal; } /* 1, 2, 3 */
ol { list-style-type: upper-roman; } /* I, II, III */

/* Custom bullet dengan ::before pseudo-element */
ul.custom li::before {
  content: "→";
  color: #2e7d32;
  font-weight: bold;
  margin-right: 8px;
}

/* Navigasi: tampilan horizontal */
nav ul { display: flex; gap: 1rem; list-style: none; }

/* list-style-position: outside (default) vs inside */
li { list-style-position: inside; } /* bullet ikut indentasi teks */

CSS Images & Filters

/* Gambar responsif — dasar */
img {
  max-width: 100%;    /* tidak overflow parent */
  height: auto;       /* pertahankan aspect ratio */
  display: block;     /* hapus whitespace di bawah gambar */
}

/* object-fit: cara gambar mengisi container */
.thumbnail {
  width: 200px;
  height: 200px;
  object-fit: cover;   /* potong gambar agar pas, pertahankan ratio */
  /* object-fit: contain — gambar penuh dalam container, mungkin ada sisi kosong */
  /* object-fit: fill — stretch, tidak pertahankan ratio */
}

object-position: center top; /* posisi fokus saat object-fit: cover */

/* CSS Filters — efek visual */
img { filter: grayscale(100%); }         /* hitam putih */
img { filter: blur(4px); }               /* blur */
img { filter: brightness(0.7); }         /* lebih gelap */
img { filter: brightness(1.3); }         /* lebih terang */
img { filter: contrast(1.5); }           /* kontras tinggi */
img { filter: sepia(80%); }              /* efek foto tua */

/* Gabungan filter */
img { filter: brightness(0.9) contrast(1.1) saturate(1.2); }

/* backdrop-filter: filter pada latar belakang (efek frosted glass) */
.glass {
  background: rgba(255,255,255,0.2);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px); /* Safari */
}
