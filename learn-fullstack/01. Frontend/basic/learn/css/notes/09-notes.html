09 · Display Property

Konsep: Setiap Elemen adalah Kotak

Properti display menentukan bagaimana sebuah elemen berpartisipasi dalam flow layout — apakah ia mengambil seluruh baris, mengalir dalam teks, atau sepenuhnya dikeluarkan dari layout. Ini adalah salah satu properti paling fundamental yang mempengaruhi segalanya.

display: block

Elemen block mengambil seluruh lebar yang tersedia dan selalu mulai di baris baru. Contoh bawaan: div, p, h1–h6, section, article. Kamu bisa mengatur width, height, dan semua properti box model.

Block elemen 1 — mengambil lebar penuh
Block elemen 2 — baris baru otomatis
/* Semua ini adalah block secara default */
div, p, h1, h2, section, article, ul, ol, li, form { display: block; }

/* Ubah inline menjadi block */
span { display: block; }

display: inline

Elemen inline mengalir dalam teks seperti kata-kata — tidak memulai baris baru, hanya mengambil ruang sebesar kontennya. width dan height tidak berpengaruh. Margin/padding kiri-kanan bekerja, tapi atas-bawah tidak mempengaruhi layout sekitarnya. Contoh bawaan: span, a, strong, em, code.

Inline 1 Inline 2 Inline 3 — mengalir dalam satu baris 
/* Inline tidak bisa diatur width/height */
span { width: 200px; } /* ← tidak ada efeknya */

/* Ubah block menjadi inline */
div { display: inline; }

display: inline-block

Gabungan terbaik dari keduanya: mengalir dalam teks seperti inline, tapi bisa diatur width, height, dan semua box model seperti block. Sangat berguna untuk badge, tombol, tag label — elemen yang perlu dimensi tapi juga harus mengalir dalam teks.

Badge HTML Badge CSS Badge JS 
.badge {
  display: inline-block;
  width: 80px;         /* bisa karena inline-block */
  padding: 4px 12px;
  background: #e8f5e9;
  border-radius: 20px;
  text-align: center;
}

display: none vs visibility: hidden

Ini adalah perbedaan yang sering membingungkan. display: none menghilangkan elemen dari layout sepenuhnya — elemen tidak ada, ruangnya diambil elemen lain. visibility: hidden menyembunyikan elemen secara visual tapi ruangnya tetap ada — layout tidak bergeser.

Kotak 1 (terlihat)
Kotak 2 (display: none — tidak ada sama sekali)
Kotak 3 — naik mengisi ruang kotak 2
Kotak 1 (terlihat)
Kotak 2 (visibility: hidden — invisible tapi ruangnya ada)
Kotak 3 — tetap di posisi semula
/* display: none — hapus dari layout */
.modal { display: none; }
.modal.aktif { display: block; }

/* visibility: hidden — sembunyikan tapi pertahankan ruang */
.placeholder { visibility: hidden; }

/* opacity: 0 — invisible tapi masih bisa diklik! */
.ghost { opacity: 0; }
⚠️ display: none juga menyembunyikan elemen dari screen reader (tidak aksesibel). Untuk aksesibilitas, gunakan teknik visually-hidden: position: absolute; clip: rect(0 0 0 0); width: 1px; height: 1px; overflow: hidden;
