12 · Pseudo Classes

Apa itu Pseudo Class?

Pseudo class adalah selector yang menarget elemen berdasarkan state atau posisi yang tidak bisa direpresentasikan oleh atribut HTML biasa. Ia menambahkan kondisi pada selector: "elemen ini, ketika dalam kondisi tertentu." Ditulis dengan awalan titik dua tunggal :.

User Action Pseudo Classes

/* :hover — mouse di atas elemen */
.btn:hover { background: #1b5e20; transform: translateY(-2px); }

/* :focus — elemen mendapat fokus (keyboard Tab atau klik) */
input:focus { outline: 2px solid #1565c0; outline-offset: 2px; }

/* :focus-visible — hanya muncul saat navigasi KEYBOARD, tidak saat klik mouse */
/* Ini lebih tepat untuk focus ring visual daripada :focus */
:focus-visible { outline: 3px solid #1565c0; outline-offset: 3px; }

/* :active — saat elemen sedang ditekan/diklik */
.btn:active { transform: scale(0.97); }

/* :visited — link yang sudah pernah dikunjungi */
a:visited { color: #7b1fa2; }

Structural Pseudo Classes

/* :first-child, :last-child — anak pertama/terakhir dari parent-nya */
li:first-child { border-top: none; }
li:last-child  { border-bottom: none; }

/* :nth-child(n) — anak ke-n dari parent */
tr:nth-child(even) td { background: #f5f5f5; } /* baris genap */
tr:nth-child(odd)  td { background: white; }   /* baris ganjil */
li:nth-child(3)    { font-weight: bold; }       /* tepat anak ke-3 */
li:nth-child(3n)   { color: green; }            /* setiap kelipatan 3: 3,6,9,... */
li:nth-child(3n+1) { color: red; }              /* 1, 4, 7, 10, ... */

/* :only-child — jika merupakan satu-satunya anak */
li:only-child { font-style: italic; }

/* :not() — negasi: elemen yang TIDAK cocok selector di dalam */
li:not(:last-child) { border-bottom: 1px solid #eee; }
input:not([type="submit"]) { border: 1px solid #ddd; }
.kartu:not(.stok-habis):hover { transform: translateY(-2px); }

Form State Pseudo Classes

/* State validasi */
input:valid   { border-color: #2e7d32; }
input:invalid { border-color: #c62828; }

/* :required dan :optional */
input:required { border-left: 3px solid #c62828; }
input:optional { border-left: 3px solid #9e9e9e; }

/* :disabled dan :enabled */
button:disabled { opacity: 0.4; cursor: not-allowed; }
input:enabled:hover { border-color: #1565c0; }

/* :checked — untuk checkbox dan radio */
input[type="checkbox"]:checked + label { color: #2e7d32; font-weight: bold; }

/* :focus-within — parent yang berisi child yang sedang fokus */
/* Sangat berguna untuk form group styling */
.form-group:focus-within label { color: #1565c0; }
.form-group:focus-within input { border-color: #1565c0; }

/* :user-invalid — seperti :invalid tapi HANYA setelah user berinteraksi */
/* Lebih ramah UX karena tidak merah saat halaman pertama dimuat */
input:user-invalid { border-color: #c62828; }

:has() — CSS Parent Selector (Modern)

/* :has() — memilih elemen yang MEMILIKI child yang cocok */
/* Ini adalah "parent selector" yang lama ditunggu-tunggu */

/* Kartu yang memiliki img di dalamnya */
.kartu:has(img) { padding: 0; overflow: hidden; }

/* Form yang memiliki input invalid */
form:has(input:invalid) .submit-btn { opacity: 0.5; cursor: not-allowed; }

/* Artikel yang memiliki setidaknya satu h2 */
article:has(h2) { border-top: 3px solid #2e7d32; }

/* Dukungan browser: Chrome 105+, Firefox 121+, Safari 15.4+ */
💡 :has() adalah salah satu fitur CSS paling powerful dalam beberapa tahun terakhir. Ia memungkinkan styling berdasarkan konten anak tanpa JavaScript. Di aplikasi retail, ini berguna untuk: styling container produk tergantung apakah ada badge diskon di dalamnya, mengubah tampilan form berdasarkan state validasi, dll.
