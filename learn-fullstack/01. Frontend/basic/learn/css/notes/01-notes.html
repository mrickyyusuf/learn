01 · CSS Introduction

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang mendeskripsikan presentasi visual dari dokumen HTML. CSS memisahkan konten (HTML) dari tampilan (CSS), sehingga satu file CSS bisa mengontrol ribuan halaman sekaligus. Tanpa CSS, browser menampilkan HTML dengan style bawaan yang sangat minimal.

Kata Cascading berarti aturan "mengalir" dari sumber umum ke spesifik — browser default → user stylesheet → author external → author internal → author inline. Yang lebih spesifik selalu menang.

Mengapa CSS Dipisahkan dari HTML?

Bayangkan sebuah toko retail dengan 500 halaman produk. Jika warna brand berubah dari hijau ke biru, dengan CSS terpusat kamu hanya mengubah satu variabel di satu file — semua 500 halaman ikut berubah. Tanpa CSS terpisah, kamu harus mengubah 500 file satu per satu.

Ini adalah prinsip Separation of Concerns: setiap lapisan hanya bertanggung jawab pada satu hal. HTML = struktur dan makna. CSS = tampilan dan presentasi. JavaScript = perilaku dan interaktivitas.

Rendering Pipeline: Dari File ke Piksel

Browser membaca HTML → membangun DOM Tree → membaca CSS → membangun CSSOM Tree → menggabungkan keduanya menjadi Render Tree → menghitung posisi elemen (Layout/Reflow) → menggambar piksel (Paint) → menggabungkan layer (Composite).

Penting: mengubah properti seperti transform hanya memicu Composite (sangat cepat). Mengubah width memicu Layout ulang (mahal). Inilah mengapa animasi dengan transform jauh lebih smooth dari animasi dengan left/top.

Anatomi CSS Rule

selector {        /* siapa yang dikenai aturan */
  property: value; /* declaration: satu pasang instruksi */
  property: value;
}

/* Contoh nyata */
.kartu-produk {
  background-color: white;   /* property: value */
  border-radius: 8px;
  padding: 16px;
}
💡 CSS bersifat deklaratif — kamu menyatakan keadaan yang diinginkan, browser yang memutuskan cara mencapainya. Berbeda dari JavaScript yang imperatif (kamu tunjukkan langkah-langkahnya).

Tiga Cara Menulis CSS

/* 1. INLINE — langsung di elemen, specificity tertinggi */
/* Hanya untuk nilai runtime dinamis (misal: width progress bar dari JS) */
<div style="width: 73%"></div>

/* 2. INTERNAL — di <style> dalam <head> */
/* Untuk halaman standalone atau critical CSS */
<style> .kartu { background: white; } </style>

/* 3. EXTERNAL — file .css terpisah, standar industri */
/* Browser cache file ini — halaman kedua dst lebih cepat */
<link rel="stylesheet" href="styles/main.css">
