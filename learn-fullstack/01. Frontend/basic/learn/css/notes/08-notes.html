CSS Units

Unit Tipe Relatif terhadap Kapan pakai
px Absolute — Border, shadow, nilai presisi
em Relative font-size elemen sendiri/parent Padding/margin relatif font
rem Relative font-size root (<html>) Typography, spacing konsisten
% Relative Ukuran parent Width/height fleksibel
vw/vh Relative Lebar/tinggi viewport Hero section, full-screen
calc() Function Campuran Width dinamis
clamp() Function Campuran Responsive typography
padding: 1em (= font-size elemen itu, sering berubah sesuai nesting)
padding: 1rem (= font-size root, selalu konsisten)

CSS Functions: calc(), min(), max(), clamp()

width: calc(100% - 48px) → selalu 48px lebih kecil dari parent
width: min(100%, 400px) → max 400px, tapi menyempit di layar kecil

font-size: clamp(1rem, 3vw, 2.5rem)

Resize jendela browser — judul di atas berubah ukuran antara 1rem dan 2.5rem.
