SOP Pembukaan Toko Setiap Pagi

  1. Datang lebih awal (±30 menit sebelum jam buka) untuk persiapan.
  2. Buka pintu dan nyalakan listrik: lampu, kipas/AC, serta perangkat elektronik kasir.
  3. Periksa stok barang di rak depan: pastikan produk tertata rapi, tidak ada kemasan rusak atau kedaluwarsa.
  4. Siapkan kasir: hidupkan komputer/mesin kasir, cek saldo uang kecil untuk kembalian.
  5. Bersihkan area toko: sapu lantai, lap meja kasir, rapikan rak agar terlihat menajar.
  6. Cek keamanan: pastikan CCTV aktif, pintu belakang terkunci, dan APAR (alat pemadam) siap.
  7. Siapkan catatan harian: tulis target penjualan atau agenda khusus hari ini.

Perlengkapan Wajib di Meja Kasir

  • Mesin kasir / komputer POS
  • Laci uang dengan pecahan kecil untuk kembalian
  • Struk printer & kertas thermal cadangan
  • Kalkulator manual (cadangan jika sistem bermasalah)
  • Pulpen & buku catatan transaksi darurat
  • Barcode scanner
  • Kantong plastik/kertas untuk belanja pelanggan
  • Hand sanitizer / tisu basah

Glosarium HTML yang Saya Ketahui

Markup
Secara bahasa berarti "penanda." Dalam konteks web dan HTML, markup adalah sistem penanda konten menggunakan tag sehingga browser tahu cara memperlakukan tiap bagian — mana yang header, mana section, mana paragraf, dan sebagainya.
XML
Extensible Markup Language — berbeda dengan HTML yang merenderkonten ke layar, XML digunakan untuk menyimpan dan mentransfer data antar sistem. Jika dibuka di browser, XML menampilkan struktur datanya apa adanya tanpa styling.
Element
Satu unit konten lengkap dalam HTML.
Terdiri dari tag pembuka, konten di dalamnya, dan tag penutup.
Contoh: <p>Ini sebuah paragraf.</p> — seluruhnya adalah satu element.
Tag
Penanda yang ditulis dalam tanda kurung siku < > yang memberitahu browser di mana suatu element dimulai dan berakhir. Hampir semua tag berpasangan: tag pembuka <p> dan tag penutup </p>.
Attribute
Informasi tambahan yang ditulis di dalam tag pembuka, selalu dalam format nama="nilai". Attribute memodifikasi perilaku atau memberi identitas pada element. Contoh: <a href="toko.html" target="_blank"> — href dan target adalah attribute.
